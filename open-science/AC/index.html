The columns in .csv files are explained in the public rmarkdown (see open methodology) while the instances are given in the following format:
  • For InterdictionMKP_instances.zip
    • One line with the number of knapsacks and number of items: \( K, N \) ;
    • One line with, for each item \(j\), its processing time \( p_j \) ;
    • One line with, for each item \(j\), its weight \( w_j \) ;
    • One line with, for each knapsack \( k \), its capacity \( C_k \).
The input data is separated by tabs or space.
